# DEVARAJ P - Cinematic Portfolio Website

A premium, cinematic personal portfolio website built with modern web technologies: Next.js 15, React 19, Tailwind CSS v4, GSAP, Three.js, Framer Motion, and Lenis smooth scrolling.

## 🚀 Features

- **Cinematic Loading Screen**: Cyberpunk style boot loader with counting percentage.
- **Interactive Three.js Particle Background**: Responsive 3D web of stars responding to mouse movements.
- **GSAP Text Animations**: High-performance scroll-triggered reveals and typewriter effects.
- **Cyberpunk Case Files Layout**: Styled dossier-inspired cards for projects, certifications, and experience.
- **Custom Cyberpunk Cursor**: Responsive hover-state cursor with trailing delays.
- **Perfect Build Quality**: Compiled using Next.js Turbopack with zero warnings or errors.

## 🛠️ Stack

- **Framework**: Next.js 15 (App Router, React 19)
- **Styling**: Tailwind CSS v4 + native CSS custom variables
- **Animations**: GSAP (GreenSock), Framer Motion, Lenis Smooth Scroll
- **3D Graphics**: Three.js

## ⚙️ Development

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
